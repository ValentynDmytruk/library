# Система управління бібліотекою

## Опис
Модульна система для автоматизації роботи бібліотеки, написана на Python. Реалізує облік книг, реєстрацію читачів та автоматичні сповіщення за допомогою патерну Observer. Центральним ядром системи є клас Library, реалізований за патерном Singleton.

## Встановлення та запуск
1. Клонувати репозиторій: `git clone <посилання_на_репозиторій>`
2. Запустити головний файл: `python main.py`
3. Запуск тестів: `python -m unittest discover tests/`
